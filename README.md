# KT-Messenger

KT-Messenger is a chat application built using Firebase using ofcourse, the Kotlin Programming languageâœ¨

# Features 
- Sign Up with phone number and verification code.
- Send messages and image files.
- Search for all contacts using the app.
- Logout feature.
- Simple and clean UI.


---
# Libraries Used
- Material Design
- Navigation Components
- Glide
- Firebase SDK
